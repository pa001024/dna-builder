# Project Overview
- Name: dna-builder (Duet Night Abyss Builder)
- Purpose: A comprehensive tool for Duet Night Abyss that provides data lookup, character build planning, growth analysis, and damage calculation.
- Main areas: web frontend, desktop app, backend server, and Rust MCP server.
- Stack summary: Vue 3 + TypeScript frontend; Pinia + Vue Router; TailwindCSS + daisyUI; Tauri 2 (Rust + WebView2); Bun + Elysia server; Drizzle ORM + GraphQL; Vitest tests; Biome formatting/linting.
- Important directories: `src/`, `server/`, `src-tauri/`, `mcp_server/`, `tools/`, `externals/dna-api/`, `public/i18n/`.
