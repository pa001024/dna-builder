## 上传信息

你可以通过如下方式上传

1. 桌面端登录皎皎角账号 然后在统计页或游戏信息页面的深渊处上传
2. 在皎皎角给指定帖子回复 (需要将角色信息设为公开)

## 隐私声明

1. 角色信息上传时，将上传你的角色信息，包括深渊阵容，角色溯源，武器熔炼等级。
2. 不会记录明文UID 但是会用SHA256对UID进行加密用于去重

## 信息查询

服务通过GraphQL查询接口提供数据查询 API端点 `https://api.dna-builder.cn/graphql`

示例请求

```graphql
query ($seasonId: Int, $charId: Int, $mainOnly: Boolean, $limit: Int) {
    abyssUsageRoleStats(seasonId: $seasonId) {
        charId
        submissionCount
        slotCount
        ownedCount
        gradeLevelDistribution
    }
    abyssUsageWeaponStats(seasonId: $seasonId) {
        weaponId
        submissionCount
        slotCount
        ownedCount
        skillLevelDistribution
    }
    abyssUsageLineupStats(seasonId: $seasonId, charId: $charId, mainOnly: $mainOnly, limit: $limit) {
        charId
        meleeId
        rangedId
        support1
        supportWeapon1
        support2
        supportWeapon2
        petId
        submissionCount
    }
    abyssUsageSlotStats(seasonId: $seasonId) {
        support {
            id
            submissionCount
        }
        meleeWeapon {
            id
            submissionCount
        }
        rangedWeapon {
            id
            submissionCount
        }
        pet {
            id
            submissionCount
        }
    }
    abyssUsageRoleRank(seasonId: $seasonId, limit: $limit) {
        charId
        submissionCount
        slotCount
        ownedCount
        gradeLevelDistribution
    }
    abyssUsageWeaponRank(seasonId: $seasonId, limit: $limit) {
        weaponId
        submissionCount
        slotCount
        ownedCount
        skillLevelDistribution
    }
}
```
